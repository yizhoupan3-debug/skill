# Assignment Compliance — Checklist Template Reference

Standard checklist template for common assignment types.

## Generic Compliance Checklist

- [ ] All required sections present
- [ ] Page/word count within limits
- [ ] Formatting matches requirements (font, spacing, margins)
- [ ] All questions/parts answered
- [ ] Citations in required format
- [ ] Submitted in correct file format
- [ ] Name/ID on all pages
- [ ] Figures/tables properly labeled and referenced

## Common Requirement Sources

| Source | Check For |
|---|---|
| Assignment PDF | Explicit requirements, rubric criteria |
| Syllabus | Formatting standards, submission rules |
| Rubric | Grading criteria, point allocation |
| Template | Required sections, structure constraints |
