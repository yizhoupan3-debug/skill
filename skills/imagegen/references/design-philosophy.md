# Design Philosophy & Visual Principles

Foundational design theory for producing museum-quality visual output.
Inspired by canvas-design principles from content-creative-super-skill.

## Design Philosophy Creation Workflow

For high-stakes visual projects, establish a **Visual Philosophy** before
producing any image. This elevates output from "nice picture" to "art direction."

### Step 1: Name the Movement (1–2 words)

Create a unique aesthetic label that guides all creative decisions.

Examples: "Brutalist Joy" / "Chromatic Silence" / "Metabolist Dreams" /
"Concrete Poetry" / "Analog Meditation" / "Neon Minimalism"

### Step 2: Write the Philosophy (4–6 lines)

Address each dimension once:

- **Form and space**: How do forms relate to negative space?
- **Color and material**: What palette governs? What textures?
- **Scale and rhythm**: Large vs small, fast vs slow
- **Composition and balance**: Symmetry, asymmetry, diagonal flow?
- **Visual hierarchy**: Where does the eye go first, second, last?

Emphasize craftsmanship — "meticulously crafted," "master-level execution."

### Step 3: Identify the Conceptual Seed

Extract the subtle, niche reference embedded in the user's request. This is
the soul woven invisibly into the art — only those who know will catch it.

### Step 4: Execute

- 90% visual, 10% essential text (text as visual accent, not explanation)
- Nothing overlaps or falls off canvas boundaries
- Typography is design-forward; never use generic fonts
- Aim for work that looks like it took countless hours by an expert

### Step 5: Refine

Ask: "How can I make what's already here more of a piece of art?"
Tighten spacing, improve color harmony, strengthen typographic choices,
deepen compositional balance. Do NOT add more shapes or graphics.

## Color Theory Quick Reference

### Color Relationships

| Relationship | Description | Effect |
|---|---|---|
| **Analogous** | Adjacent on color wheel | Harmonious, low contrast |
| **Complementary** | Opposite on wheel | High contrast, vibrant tension |
| **Triadic** | Three evenly spaced | Balanced, energetic |
| **Split-complementary** | One + two adjacent to complement | Softer than complementary |
| **Monochromatic** | One hue, varying saturation/value | Sophisticated, cohesive |

### 60/30/10 Distribution Rule

- **60%** — Dominant color (backgrounds, large areas)
- **30%** — Secondary color (supporting elements)
- **10%** — Accent color (CTAs, highlights, small pops)

Dominant colors with sharp accents outperform timid, evenly-distributed palettes.

## Typography Pairing Rules

### Principles

- Pair a distinctive **display font** (headings) with a refined **body font**
- Weight contrast: bold display vs regular body creates hierarchy without color
- Limit to 2 font families per composition (3 maximum for complex layouts)

### Blacklist (never use as primary display)

Arial, Inter, Roboto, Helvetica Neue, system-ui, Calibri, Times New Roman

### Strong Pairings

| Display | Body | Vibe |
|---------|------|------|
| Playfair Display | Source Serif Pro | Classic editorial |
| Clash Grotesk | Instrument Serif | Modern contrast |
| Bebas Neue | DM Sans | Bold & clean |
| Cormorant Garamond | Lato | Elegant & readable |
| Space Grotesk | IBM Plex Sans | Technical & modern |

## Composition Principles

- **Rule of thirds**: Place key elements at grid intersections
- **Visual weight**: Larger, darker, more saturated = heavier
- **Negative space**: 40-60% breathing room; avoid cluttered layouts
- **Alignment**: Every element aligns to a grid or another element
- **Hierarchy**: One dominant element, one secondary, everything else subordinate
- **Visual anchor**: Main element centered or offset left; reserve right for title

## Anti-Tacky Checklist

Before finalizing any visual output, verify:

- [ ] No stock-photo vibe
- [ ] No cheesy lens flare or harsh bloom
- [ ] No oversaturated neon (unless intentional aesthetic)
- [ ] No oversharpening
- [ ] No clutter — every element earns its place
- [ ] Color palette is intentional, not arbitrary
- [ ] Typography choices are deliberate, not default
- [ ] Composition has clear hierarchy
- [ ] Generous whitespace where needed
