# Revision Modes

Use these modes inside `paper-reviser`, not as separate top-level owners.

- `logic repair`: narrow, reframe, delete, or add evidence requirements for unsupported claims.
- `notation cleanup`: apply notation and abbreviation fixes after a sweep.
- `length budget`: cut, move to appendix, compress tables, or expand only where evidence supports it.
- `figure-table repair`: update captions, labels, table density, or call plotting owners for code-level redraws.

Every revision should state whether the blocker is resolved, partially resolved,
or still blocked.
