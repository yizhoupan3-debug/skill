# Citation style and source policy

## Source trust policy

Use this order when resolving canonical metadata:
1. Publisher page
2. DOI / Crossref record
3. PubMed / PMC where relevant
4. Official journal or library metadata
5. Author-hosted accepted version
6. Preprint server only as fallback

## Normalization checklist

Before style conversion, confirm:
- canonical title
- canonical author order
- year
- venue type
- volume / issue / pages or article number
- DOI / PMID / URL as applicable
- whether a peer-reviewed version supersedes a preprint

## Style conversion reminders

### APA
- author-date system
- sentence case for article titles
- italicize journal title and volume when formatting supports it
- include DOI in normalized URL form when available

### IEEE
- numeric bracket citations
- abbreviated first names / initials as required
- article title in quotation marks
- journal or proceedings title in italic when formatting supports it

### ACM
- author-date or numbered variants depend on venue template; do not assume without checking
- preserve DOI when available
- normalize venue names carefully because ACM references are template-sensitive

### GB/T 7714
- distinguish 文献类型 when needed
- pay attention to author truncation, punctuation, and carrier/type markers
- confirm whether the target venue expects 顺序编码制 or 著者-出版年制 before rendering

## Precision heuristics for citation placement
- Prefer one directly supporting citation over a loose citation cluster.
- Split sentences when different claims need different sources.
- Use survey citations for broad landscape claims, not for specific benchmark or method details when a primary source is available.
