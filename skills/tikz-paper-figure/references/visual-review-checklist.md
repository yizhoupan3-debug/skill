# Visual Review Checklist

Use the rendered PNG/PDF, not only the source code.

## Required Checks

- The PDF is cropped and has no large paper-like margins.
- The figure still reads at the intended paper width.
- All labels are legible and aligned.
- No label, formula, or legend touches a border.
- No line crosses text.
- No arrowhead points the wrong way.
- No connector pierces the inside of a box.
- Same-level elements are consistently sized.
- The main module or message has clear visual priority.
- Background zones are subtle and do not compete with content.
- Dashed feedback lines are visually distinguishable from zone boundaries.
- There is no caption, figure number, prompt text, watermark, or AI metadata
  inside the figure canvas.

## Fix Order

1. Fix compile errors.
2. Fix missing or clipped content.
3. Fix wrong arrow semantics.
4. Fix overlaps and line crossings.
5. Fix text size and paper-width readability.
6. Fix hierarchy, whitespace, and final polish.

Do not polish color before structural problems are solved.
