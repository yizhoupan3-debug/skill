# Design Audit Rubric

## 1. Atmosphere Match

- 输出整体气质是否还在原来的频道里
- 密度、留白、节奏是否偏离

## 2. Palette Discipline

- 主次色是否还守规矩
- 有没有多余 accent、脏渐变、乱跳色温

## 3. Typography Hierarchy

- 标题、正文、说明、数字的层级是否清楚
- 有没有回退成泛默认字体或无差别字号

## 4. Component Signature Match

- 按钮、卡片、输入框、导航是否还像同一套系统
- 是否出现“每块都长得像不同产品”这种碎裂感

## 5. Layout Grammar

- 栅格、对齐、留白、分区节奏是否一致
- 有没有偷懒回到平均分三列、挤满卡片、中心大堆叠

## 6. Anti-Pattern Relapse

- 有没有重新出现禁区里的东西
- 例如 AI 紫光、假数据卡片、空泛大词、风格污染、无意义装饰

## Verdict Levels

- `pass`: 基本守住系统
- `minor drift`: 有偏移，但局部修正即可
- `material drift`: 风格开始散，需要一轮有目的的修正
- `hard fail`: 系统已经失真，必须回到设计约束重新生成或重做
