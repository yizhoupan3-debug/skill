# Review Dimensions

Use these as modes inside `paper-reviewer`, not as separate top-level owners.

- `logic mode`: claim ceiling, novelty, evidence coverage, ablation isolation, comparison fairness.
- `language / readability`: field-standard terms (no undefined coinages), terminology density, repetition without new evidence, no stacked defensive contrast (`but` / `not` / `rather than`), no internal-only phrasing, no pasted code identifiers or bare `results.csv`-style pointers instead of tables/figures/supplement numbering; see [`../../paper-workbench/references/research-language-norms.md`](../../paper-workbench/references/research-language-norms.md).
- `notation sweep`: abbreviation first use, symbol uniqueness, equation numbering, cross-references, units.
- `figure-table mode`: rendered figure/table readability, caption self-containment, axis/legend clarity.
- `length risk`: whether page pressure hides missing evidence or forces appendix routing.

Return verdict first, then the few blockers most likely to affect acceptance.
