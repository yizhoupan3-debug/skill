# Eval Fixture: PPT Project

This is a minimal fixture workspace used by the behavioral eval harness.
It contains a simplified PPT build script so the eval runner can verify
that the agent reads the right files and produces the right artifacts.

## Structure

- `ppt/build_deck.sh` — minimal deck builder
- `ppt/extracted_images/` — placeholder image directory
- `ppt/case_study_images/` — placeholder case study directory
