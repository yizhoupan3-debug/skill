# Routing eval cases

Canonical routing eval corpus: **`tests/routing_eval_cases.json`** (schema `routing-eval-cases-v1`).

`router-rs eval route --cases <path>` accepts that file. The former `routing.json` duplicate in this directory was removed to keep a single source of truth.

Field-level documentation: `evals/SCHEMA.json`（`routing_eval_cases_v1` 与 `routing_eval_report_v1`）。
